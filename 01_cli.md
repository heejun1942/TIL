# CLI

커맨드 라인 인터페이스



## 터미널 명령어들

- `ls`: 폴더 내부의 파일&폴더를 나열
  - `ls -a`: 숨김폴더까지 보여줌
- `pwd`: 현재 폴더 경로 출력
- `mkdir [폴더명]`: 폴더 생성
- `cd [폴더명]`: (change directiory) 폴더 변경
  - `cd ..`: 상위 폴더로 나감
- `git init`: git으로 프로젝트 생성
- `git status`: git의 현재 상태 출력
- `touch [파일명]`: 파일 생성
- `rm [파일명]`: 파일 삭제
